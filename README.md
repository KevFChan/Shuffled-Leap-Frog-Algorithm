## ğŸ“Œ Overview
The **Shuffled Frog Leaping Algorithm (SFLA)** combines concepts from both **genetic algorithms** and **particle swarm optimization**. A population of frogs is partitioned into memeplexes, which evolve independently through local search before being periodically shuffled to exchange knowledge and prevent premature convergence.

For this project, I explained why the SFLA is valid, analyzed the global convergence of the algorithm using Markov chains, and compared the superior performance against other established algorithms in the context of neural network training.
